# Algorithms Library Wiki

Welcome to the project wiki for the Algorithms Library. This library provides
JavaScript implementations of common algorithms and data structures, organized
by domain.

## Quick Links

- Getting Started: [Getting-Started](Getting-Started.md)
- Algorithms: [Algorithms](Algorithms.md)
- Data Structures: [Data-Structures](Data-Structures.md)
- Usage Examples: [Examples](Examples.md)
- Notes and Conventions: [Notes-and-Conventions](Notes-and-Conventions.md)

## Project Layout

- Source code lives under `src/`.
- Algorithms are in `src/algorithms/`.
- Data structures are in `src/dataStructures/`.
- The public entry point is `src/index.js`.
